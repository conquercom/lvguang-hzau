# 绿光华农 🌱

华中农业大学环保积分平台 - 让每一次环保行为都值得被看见

## 项目简介

绿光华农是一款面向华中农业大学师生的环保积分Web应用，鼓励用户记录绿色行为、获取积分、兑换奖励。

## 功能特色

- 🥢 **绿色打卡** - 拒绝一次性餐具、随手关灯、节约用水等
- 🎁 **积分兑换** - 图书馆研讨室、体育馆预约、食堂优惠券等华农特色奖励
- 🏆 **排行榜** - 个人榜、学院榜，激励环保行为
- 📊 **数据统计** - 可视化展示节能减排成果
- 🛡️ **信誉系统** - 防作弊机制，保障公平

## 技术栈

- React 18 + Vite
- TailwindCSS
- React Router
- Lucide Icons

## 本地运行

```bash
npm install
npm run dev
```

## 构建部署

```bash
npm run build
```

构建产物在 `dist` 目录

- 桃园、荟园、博园食堂
- 狮子山骑行打卡
